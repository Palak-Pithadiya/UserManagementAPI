# User Management API
This project is the final assignment for the **Back-End Development with .NET** course on Coursera.

## Features
* **Full CRUD:** Manage users with GET, POST, PUT, and DELETE endpoints.
* **Validation:** Uses System.ComponentModel.DataAnnotations to validate user input.
* **Middleware:** Includes a custom logging middleware to track API requests in the console.
* **AI Assisted:** Developed and debugged using GitHub Copilot.

## How to Run
1. Clone the repository.
2. Run `dotnet run` in the terminal.
3. Access the API at `http://localhost:5000/users`.
